# 🚨 RideWatch

**A Ride Safety Web Application for Commuters**

RideWatch is a React-based web app built to improve personal safety for commuters. It provides emergency alerting, contact management, and real-time notifications — all in a clean, mobile-friendly interface.

---

## ✨ Features

- **SOS Emergency Alert** — Sends an email alert with the user's live GPS coordinates via EmailJS and the browser Geolocation API
- **Emergency Contacts Manager** — Add, edit, and manage personal emergency contacts with shared state across components
- **Notifications Center** — View and manage safety-related notifications in one place
- **Tab-Based Navigation** — Smooth SPA experience with React Router and a persistent layout

---

## 🛠 Tech Stack

- **Frontend:** React, Vite, React Router DOM
- **Email Integration:** EmailJS
- **Geolocation:** Browser Geolocation API
- **Styling:** Custom CSS
- **Version Control:** Git & GitHub
- **Deployment:** *(in progress)*

---

## 🚀 How to Run Locally

1. Clone the repo:
```bash
   git clone https://github.com/Icession/RideWatch.git
```
2. Install dependencies:
```bash
   npm install
```
3. Run the dev server:
```bash
   npm start
```
